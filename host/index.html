WePush API 接口文档

状态：● 核心服务运行正常

这是一个私有的微信公众号/测试号消息推送接口，支持 GET 和 POST JSON 调用。

🚀 快速调用示例 (GET)

GET https://we.nn.kg/api.php?key=YOUR_KEY&title=Hello&content=World

📦 Payload 参数说明

  • key: (必填) 接口鉴权密钥
  • title: (可选) 消息标题，默认"通知"
  • content: (可选) 消息主体内容
  • url: (可选) 点击消息后的跳转链接
